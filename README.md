This is a small project made by me. Hope you like it.

To use the project, follow these steps:
  - Start the server: Enter this command [python -m uvicorn main:app --port 8080 --reload]
  - Run the server: Paste this on your browser [http://localhost:8080]

Remember, the website will run completely offline after downloading it from my repository.
If you have a weak computer, it is recommended to create a .env file inside the main project folder where you paste REMOVEBG_API_KEY=[your API key here], replace [your API key here] with your real API key
